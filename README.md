# Js-for-Beginners
JavaScript로 프로그래밍에 입문하려는 지인들을 위한 리포지토리

## 🛠️ 실행 환경 및 코드 버전

### ES6
이 저장소의 코드 예제들은 최신 JavaScript ES6을 기준으로 한다
코드에서는 `var`가 사용되었으나, ES6버전 이후로 `let`과 `const` 사용을 권장

### Node.js
해당 리포지토리에 존재하는 대부분의 코드들은 실행하기 위해서 `node.js`를 필요로 하니 `node.js` 설치 필수
